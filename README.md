# SyncDrop - Secure P2P File & Code Sharing

SyncDrop is an ultra-fast, highly secure, decentralized peer-to-peer (P2P) file and code sharing application. It enables frictionless transfers between devices (like laptops and phones) directly over the local network or internet using WebRTC, without relying on cloud storage or intermediate servers.

## ✨ Features

- **Direct P2P Transfers:** Files are sent directly between devices using WebRTC. No servers, no speed limits, no file size limits.
- **Modern Glassmorphic UI:** A beautifully crafted, fully responsive design with dynamic backgrounds and micro-animations.
- **Cross-Platform:** Works on any modern web browser across Windows, macOS, Linux, iOS, and Android.
- **Seamless QR Code Pairing:** Instantly connect your phone to your computer by scanning a dynamically generated QR code right from your screen.
- **Code & Text Sharing:** Share code snippets with automatic syntax recognition and single-click copy functionality.
- **End-to-End Encrypted:** WebRTC guarantees that your data is encrypted in transit.

## 🚀 How to Use

1. **Open SyncDrop:** Open the application on your first device (e.g., your laptop).
2. **Scan or Enter Code:** Open the app on your second device (e.g., your phone) and either tap the "Scan QR" button or manually enter the 6-character connection code displayed on your first device.
3. **Share:** Once connected, the interface will change. You can now drag and drop files, select photos, or paste code to instantly send it to the other device.

## 🛠️ Technology Stack

- **Core:** HTML5, CSS3, Vanilla JavaScript
- **Networking:** PeerJS (WebRTC Wrapper)
- **Utilities:** HTML5-QRCode (for seamless camera scanning), QRCode.js (for generating connection codes)
